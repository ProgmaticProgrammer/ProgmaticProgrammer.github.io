Qt Automotive Suite comes with components and tools that assist you to develop, deploy, diagnose, and debug your system.

Qt Application Manager

Application Manager provides the QML run-time environment where the system UI is executed. It includes four parts:

  • Application Manager, for launching, stopping and controlling applications.
  • Application Installer, for installing, updating and removing applications.
  • Window Manager, for implementing a Wayland compositor.
  • Notification Manager, for implementing org.freedesktop.Notification protocol.

Qt IVI and QFace

Qt IVI provides C++ and QML interfaces to access vehicle features and infotainment middleware. It also enables you to auto-generate code to implement new IVI features.
